# tufty2350

## What? Why? Who?
My growing collection of apps for the Tufty2350

*Warning* Some of these files have adult language though granted they are partially obscured

## How do I use these files?

- Download the desired app
- Open your Tufty in the usual manner
- Make a new folder with the name of the App e.g `F-Meter`
- Rename the downloaded file to `__init__.py`
- Upload the file to the newly created folder
- Add an icon.png to the newly created folder
